## 🛠️ 本地构建

### 环境要求

- Ruby (建议 3.2.0 或更高版本)
- Bundler

### 安装依赖

```bash
gem install bundler
bundle install
```

### 生成 HTML 版本

```bash
rake html
```

### 生成 PDF 版本

```bash
rake pdf
```

注意：PDF 生成需要安装 wkhtmltopdf

## 📂 项目结构

- `text/` - 英文原版内容
- `text_zh/` - 中文翻译内容
- `layout/` - 页面布局模板
- `script/` - 构建脚本
- `output/` - 生成的输出文件
